An integrated four-application pipeline for automated thermodynamic modelling along subduction geotherms — from raw geochemical data to redox analysis across thousands of P-T paths.

01 ThermoChemX Bulk & redox setup from oxide wt.% data

Built for
systematic

ThermoPathX integrates with PerpleX (Connolly, 2005) and the van Keken & Wilson (2023) subduction zone database to enable large-scale thermodynamic modelling that was previously impossible to run manually.

  • 30,000+ subduction geotherms across 56 zones
  • DEW19HP and DEW24HP thermodynamic databases
  • 1D fractionation and 2D gridded minimisation
  • Batch processing — 394 models in under 2 hours
  • Full provenance tracking on every output file
  • Windows & macOS · Open source (MIT)
